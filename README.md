# Undefined Core
This is the core mod fora all my mods

## Add it to your project
##### build.gradle.kts
```kotlin
repositories {
  maven("https://jitpack.io")
}

dependencies {
  modCompile(group = "com.github.Ricky12Awesome", name = "UndefinedCore", version = "1.1.0")
}
```
